# APP DIRETORIA VIMACOM

Estrutura pronta para subir no GitHub Pages.

## Pastas
- `index.html`: Home Diretoria
- `financeiro/index.html`: módulo Financeiro
- `vendas/index.html`: módulo Vendas
- `compras/index.html`: módulo Compras
- `assets/`: ícones e logo
- `manifest.json`: configuração PWA
- `sw.js`: service worker

## Publicação
Suba todo o conteúdo desta pasta na raiz do repositório GitHub Pages.

A Home abre os módulos por links relativos:
- `./financeiro/`
- `./vendas/`
- `./compras/`

Nos módulos, o botão Painel Executivo volta para `../`.
